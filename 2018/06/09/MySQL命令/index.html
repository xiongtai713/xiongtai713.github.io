MySQL命令

2018-06-09 | 阅读：次

#命令 进入 mysql -u root -p 输入密码

忘记密码

  1. 点击系统偏好设置->最下边点MySQL，在弹出页面中，关闭服务
  2. cd /usr/local/mysql/bin/
  3. sudo su
  4. ./mysqld_safe –skip-grant-tables &
  5. 回车后mysql会自动重启，在偏好设置中mysql的状态会变成running
  6. ./mysql
  7. FLUSH PRIVILEGES;
  8. SET PASSWORD FOR ‘root’@’localhost’ = PASSWORD(‘新密码’);

mysql 的dos命令

  1. 查询数据库的版本

    select version();

  2. 查询当前的时间

    select now();

  3. 查询当前的数据库

    show databases;

  4. 创建数据库：

    create database if not exists go default charset utf8 collate utf8_general_ci;

    若报错 ERROR 1044 (42000): Access denied for user ''@'localhost' to database 'day0511' 是权限问题，需 mysql -u root -p 进入MySQL。

  5. 切换数据库

    use day0510(数据库名)

  6. 查询当前操作的数据库

    select database();

  7. 删除数据库

    drop database if exists day0511(数据库名);

  8. 查看该数据库下的数据表

    show tables;

  9. 创建数据表

    create table if not exists users( 
id int(4) primary key auto_increment not null,
age int(4) default 20,
price float,
name varchar(20) unique not null,
password varchar(30) not null );

    约束：

    1. 主键约束：primary key

      主键约束：要求该字段数值不允许空，而且数值唯一。(not null+unique)

      主键约束用于唯一的标识表中的每一个行记录的。一个表中最多只能有一个主键约束。

      主键自增： auto_increment

      要求该字段数值不允许空，而且数值唯一。所以我们通常会指定主键类型为整型，然后设置其自动增长，这样可以保证在插入数据时主键列的唯一和非空特性。

      删除主键：

      alter table stu drop primary key;

      添加主键：

      alter table teachers add primary key(id);

      修改表时设置主键自动增长：

      alter table stu modify sid  int auto_increment;

      修改表时删除主键自动增长：

      alter table stu modify sid  int;

    2. 默认值：default

    3. 非空约束：not null

      如果某个字段添加了非空约束，那么该字段的数值不能为null，对于mysql而言，表示什么都没有，未知的。不是0，也不是空”“，更不是”NULL”。

      删除非空约束：

      alter table stu modify sname varchar(20);

      添加非空约束：

      alter table users modify name varchar(20) not null;

    4. 唯一性约束：unique

    唯一性约束强调所在的字段所有的数值都是唯一的。不允许重复。允许空值（多个）。

    删除唯一约束：

       alter table users modify name varchar(20);

    添加唯一约束：

       alter table users modify password varchar(30) unique;

    1. 外键约束：foreign key（略）

  10. 查看表结构

desc users(表名);

  1. 查看表数据

    select * from users;

  2. 查看建表时的语句

    show create table users;

  3. 增加字段

    alter table users add(
note varchar(40),
rtime date);

  4. 修改字段类型

    注意事项：

    A：如果数据表中没有数据(空表)，可以任意修改字段类型

    B：如果数据表中有数据，只能修改兼容类型，注意长度。

    alter table users modify password char(20);

  5. 更改字段的名字

    alter table users change password pwd char(20);

  6. 更改表名

    rename table users to user2;

  7. 删除表

    drop table if exists user2;

    ​

修改数据

  1. 插入数据：

    A：全列插入

    insert into 表名 values(v1，v2，v3….)

    Values后数值的个数，必须和字段的个数一致。

    insert into user2 values(0,’三胖’,’123456’,’有点胖’,’2018-04-18’);

    B：缺省插入

    insert into 表名(字段1，字段2，字段3…)values (v1，v2，v3.。。)

    自增的主键，自动添加数据。

    insert into user2(username, note) values(‘老二’,’傻傻的’);

    C：同时插入多条数据

    insert into users
values
(0,'tom','tom666','猫','2018-03-21'),
(0,'jerry','memeda','mouse','2017-10-10');

    insert into user2
(name,pwd)
values
('jessica','987654'),
('如花','654654');

  2. 更新数据

    update 表名 set [更新的数据] where [更新的条件]

    update users set username=’老四’ where uid=4;

    update users set note=’酷酷的’ where username<>jessica;

    update users set note=’大学生’ where age between 20 and 25;

    update users set rtime=’2015-10-10’ where uid in(1,4,6);

    update users set rtime=’2017-09-09’ where age=26 and note=’酷酷的’;

    update users set rtime=’2013-1-14’ where pwd is NULL;

  3. 删除数据

    delete from 表名 where [删除条件]

    delete from users where uid=6;

  4. 查询数据

    select 列名，列名，列名。。。From 表名

    select * from users;

    select age from users;

  5. 条件查询

    select * from users where note=’大学生’;

    select age from users where note=’大学生’;

    select * from users where age=22 or note=’酷酷的’;

    select * from users where age>=25 and age<=28;

    条件是可选的

    1. 条件必须是boolean类型的值或者表达式
    2. 运算符：=,!=,<>,>,<,>=,<=,between..and, in(…),or ,and ,not, is null，is not null
    3. is null 不是 = null(永远是false)

  6. 模糊查询

    %通配符代表任意多个字符，_通配符代表任意一个字符

    select * from users where note like(模糊关键字) ‘%学’;

    select * from users where note like(模糊关键字)’%生’;

    select * from users where note like(模糊关键字) ‘%酷%’;

    select * from users where username like(模糊关键字) ‘老_’;

  7. 排序

    select * from users order by(关键字) age asc; (从小到大)

    select * from users order by age desc; (从大到小)

    select * from users order by age,uid desc;

  8. 统计函数（聚合函数）

    select count(pwd) from users(表名); (总数)

    select max(age) from users; (最大)

    select min(age) from users; (最小)

    select sum(age) from users; (求和)

    select avg(age) from users; (求平均值)

  9. 分页查询

    select * from users limit 2,5;

  10. 去重查询

select distinct note from users;
