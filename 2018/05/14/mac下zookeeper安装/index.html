#安装zookeeper

1.0 前提工作：

安装jdk 配置环境变量，自行百度

jdk下载地址：http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

###1.1 查看zookeeper信息

$ brew info zookeeper

###1.2 安装zookeeper

  $ brew install zookeeper

###1.3 启动服务

$ zkServer
$ zkServer status
$ zkServer start

###1.4 查看zookeeper运行状态

$ zkCli
